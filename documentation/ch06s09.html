6.9.1. Description

Processing categories: point

The Mrgelev merges multiple ASCII elevated files created by Smkmerge. The program can also output a binary elevated file needed by CAMx and UAM models. Mrgelev also handles matching PinG sources from different inputs, ensuring that only the top 300 emission sources are marked as PinG sources in the final merged output. For PinG matching, Mrgelev uses PELV files output by Elevpoint. Mrgelev uses the environment variable MRG_DIFF_DAYS to merge files from different days into a single file.
