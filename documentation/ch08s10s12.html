8.10.12. SICDESC: SIC code descriptions

This file provides text descriptions of the SIC codes. There are delimited and fixed formats available for the SICDESC file shown in Table 8.98, “Delimited format for SICDESC” and Table 8.99, “Fixed format for SICDESC”. 

#DELIMITED
"0111","Agricultural Production - Crops, Cash Grains, Wheat"

Table 8.98. Delimited format for SICDESC

Position Name Type Description
A CODE Char (20) SIC code
B DESC Char (200) Description for SIC code 
#FIXED
0111 Agricultural Production - Crops, Cash Grains, Wheat

Table 8.99. Fixed format for SICDESC

Position Name Type Description
1-4 CODE Char (20) SIC code
5-205 DESC Char (200) Description for SIC code
