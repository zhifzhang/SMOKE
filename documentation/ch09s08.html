The Met4moves program is responsible for creating inputs for the MOVES Driver Script and the SMOKE Movesmrg program. Met4moves creates the following intermediate files:

  • Section 9.8.1, “MOVES_OUTFILE”

  • Section 9.8.2, “MOVES_RH_OUTFILE”

  • Section 9.8.3, “SMOKE_OUTFILE(=METMOVES)”

9.8.1. MOVES_OUTFILE

Type: ASCII - column-specific

Used by: MOVES Driver Script

The output file created by Met4moves for RPP mode in the MOVES driver script contains the absolute minimum and maximum temperatures, and sets of 24 hour diurnal temperature profiles based on combinations of min/max temperature bins for all representative counties by temperature bin.

Table 9.20. File columns in MOVES_OUTFILE

Column Type Units Description
A Integer Unitless Representative County
B Integer Unitless Fuel Month
C Character Unitless temperatureProfileID
D Real Fahrenheit Temperature at hour 0
E Real Fahrenheit Temperature at hour 1
F Real Fahrenheit Temperature at hour 2
G Real Fahrenheit Temperature at hour 3
... ... ... ...
AC Real Fahrenheit Temperature at hour 23
