6.17.2. Processing Order

The Normbeis3 program must be executed before Tmpbeis3.

If you are using the seasonal switch option, the seasonal switch file (BIOSEASON) must be created using the utility program Metscan before running Tmpbeis3.

Tmpbeis3 must be executed before biogenic emissions can be used in Smkmerge.
