8.10.11. SCCXREF: Reference SCC input file

This file provides mapping between reference SCC (i.e., SCC8) and full SCC (i.e., SCC10) for Smkinven to convert reference SCC-level inventory to full SCC inventory:

Table 8.97. Delimited format for SCCXREF

Lines Position Description
1+ A Full SCC (20-character maximum string)
  B Reference SCC (20-character maximum string)
