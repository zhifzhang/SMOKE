8.10.5. MACTDESC: MACT code descriptions

This file provides textual descriptions of the MACT codes. The format of the MACTDESC file is shown in Table 8.90, “Format for MACTDESC”.

Table 8.90. Format for MACTDESC

Position Name Type Description
A CODE Char (6) MACT code
B DESC Char (200) Description for MACT code
