9.5. Gentpro

9.5.1. TPRO_MON
9.5.2. TPRO_DAY
9.5.3. TPRO_HOUR

The Gentpro program is responsible for creating the monthly, daily or hourly temporal profiles for Temporal. Gentpro creates the following intermediate files depending on the options:

  • Temporal cross-reference file: Section 8.3.2, “[A|M|P]TREF: Temporal cross-reference file”

  • Monthly temporal profile file: Section 9.5.1, “TPRO_MON”

  • Daily temporal profile file: Section 9.5.2, “TPRO_DAY”

  • Hourly temporal profile file: Section 9.5.3, “TPRO_HOUR”

9.5.1. TPRO_MON

Type: ASCII - list-directed

Used by: Temporal
