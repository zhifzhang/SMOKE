6.10.2. Processing Order

Mrggrid can be used with any gridded output files from Smkmerge. It can also be used with other I/O API NetCDF gridded files produced outside of SMOKE.

Output files from Mrggrid are usually moodel-ready input files that will be used in running an AQM.
