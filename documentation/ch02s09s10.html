2.9.10. Convert coordinates from UTM to lat-lon

The ORL and FF10 point-source formats permit stack coordinates to be provided in UTM or lat-lon coordinates, and the SMOKE intermediate inventory stores the coordinates as lat-lon values. Smkinven converts the UTM coordinates to lat-lon coordinates using the I/O API routine UTM2LL. Lat-lon coordinates must be provided in decimal degrees, while UTM coordinates must be provided in meters.
