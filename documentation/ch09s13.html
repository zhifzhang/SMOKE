9.13. Spcmat

9.13.1. ASMAT_S, MSMAT_S, and PSMAT_S
9.13.2. ASMAT_L, MSMAT_L, and PSMAT_L

The Spcmat program is responsible for creating speciation matrices. Spcmat creates the following intermediate files:

  • Speciation matrix file (mass-based): Section 9.13.1, “ASMAT_S, MSMAT_S, and PSMAT_S”

  • Speciation matrix file (mole-based): Section 9.13.2, “ASMAT_L, MSMAT_L, and PSMAT_L”

9.13.1. ASMAT_S, MSMAT_S, and PSMAT_S

Type: I/O API NetCDF - adapted gridded format

Used by: Smkmerge and Smkreport
