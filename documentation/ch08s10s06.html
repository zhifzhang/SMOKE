8.10.6. NAICSDESC: NAICS code descriptions

This file provides text descriptions of the NAICS codes. The format of the NAICSDESC file is shown in Table 8.91, “Format for NAICSDESC”.

Table 8.91. Format for NAICSDESC

Position Name Type Description
A CODE Char (6) NAICS code
B DESC Char (200) Description for NAICS code
