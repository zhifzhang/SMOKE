8.4.6. GRIDDESC: Grids and projection information

The GRIDDESC file is used to provide grid and projection information to SMOKE and can contain data for many different projections and many different grids. The format of the GRIDDESC file is available in the I/O API documentation at http://www.baronams.com/products/ioapi/GRIDDESC.html.
