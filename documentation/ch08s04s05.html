8.4.5. BGPRO: Biogenic spatial surrogate file

The BGPRO file must be provided to SMOKE when using county land use data, in order to provide a means of converting county land use to gridded land use. The format for the BGPRO file is the same as for the Area and Mobile spatial surrogate files (see Section 8.4.2, “Area and Mobile Spatial Surrogates Files”).
