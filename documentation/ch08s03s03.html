8.3.3. HOLIDAYS: Holidays date list

SMOKE can process holidays as specific days of the week in order to better model activity patterns. The HOLIDAYS file is used to indicate which days should receive special treatment and how the days should be treated. The format of this file is shown in Table 8.35, “Format for HOLIDAYS”.

Table 8.35. Format for HOLIDAYS

Line Position Description
1+ A Five-digit FIPS code, only 00000 is currently supported (required)
  B Month of holiday (1 - 12) (required)
  C Day of holiday (1 - 31) (required)
  D Year of holiday (4 digits) (required)
  E Day of the week to treat holiday as, valid values are Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, and Sunday (required)
