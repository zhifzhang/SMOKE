6.14.2. Processing Order

All other SMOKE core programs except the optional program Mrggrid prepare input for the Smkmerge program. Smkmerge is the last nonoptional SMOKE program to be run.

Individual output files from Smkmerge can be merged using Mrggrid if desired.
