8.10.9. RWC_COUNTY_TEMP: County-specific minimum temperature for Residential Wood Combustion (RWC) Equations

This file allows to county-specific minimum temperature that is required for RWC equations listed at Section 5.3.5, “Gentpro” . The format of the RWC_COUNTY_TEMP file is shown in Table 8.94, “Format for RWC_COUNTY_TEMP”.

Table 8.94. Format for RWC_COUNTY_TEMP

Position Name Type Description
A FIPS Char(5) FIPS code
B Value Real Minimum temperature
