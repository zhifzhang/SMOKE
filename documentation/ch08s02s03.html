8.2.3.1. FF10 Format

8.2.3.1. FF10 Format

The user specifies the data provided using this command in the header: FF10_HOURLY_[NONPOINT|POINT|NONPOINT|ONROAD].

This FF10 format uses the header described in Section 8.2.1.4, “Header records”. Sample header records for daily nonpoint sources are shown below: 

#FORMAT   FF10_HOURLY_NONPOINT
#COUNTRY  US
#YEAR     2005
#DESC     Nonpoint Hourly Source Inventory
#DESC     FF10 Nonpoint Hourly format
