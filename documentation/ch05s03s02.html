5.3.2.1. Description

Aggwndw aggregates and/or windows emissions from an I/O API file to a new grid. The program does not change grid projections or interpolate data; the grid boundaries of the input and output grids must be aligned and the output grid must be entirely contained within the input grid. Aggwndw automatically checks that the input and output emissions do not differ by more than 0.01% and also produces a report file containing summed input and output data.

5.3.2.2. Processing Order

Aggwndw can be run on any gridded I/O API file. Usually it would be used to create coarse-grid or windowed emissions from fine-grid model-ready output files created by Smkmerge or Mrggrid.
