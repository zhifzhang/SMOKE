10.2.4. ACSUMREP, MCSUMREP, and PCSUMREP

Type: ASCII - list-directed

Summary: This file contains all of the control (not growth) information applied to each source, by source. The structure of this file is not amenable for import into a spreadsheet, because all control information applied to each source is listed underneath each source (not on the same line). There is no header in this file.
