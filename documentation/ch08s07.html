8.7.1. FILELIST: Gridded merge 2-D file list

The FILELIST file is used to provide a list of logical file names to the Mrggrid program. This list should contain the logical file names for all 2- or 3-D files to be merged in a given run. It is a very simple file, and has no limit to the number of files listed. Note that unlike most list files used in SMOKE, FILELIST requires a list of logical file names, not the actual file name.
