10.4. Mrggrid

10.4.1. OUTFILE

The Mrggrid program is responsible for creating model-ready emissions files. Mrggrid creates the following output file:

  • Model-ready emissions file: Section 10.4.1, “OUTFILE”

10.4.1. OUTFILE

Type: I/O API NetCDF - adapted gridded format

Summary: Final, model-ready file created from two or more souce categories. See Section 10.6, “Smkmerge” for file format.
