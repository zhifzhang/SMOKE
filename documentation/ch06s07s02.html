6.7.2. Processing Order

Met4moves must be run prior to MOVES processing scripts. The SMOKE Utility program, Metcombine, needs to be run prior to Met4moves. Metcombine allows the user to combine ambient temperature, pressure and mixing ratio variables from two different MCIP files (METCRO_2D and METCRO_3D) that are needed to compute Relative Humidity (RH) in Met4moves.
