9.8.3. SMOKE_OUTFILE(=METMOVES)

Type: I/O API NetCDF - adapted gridded format

Used by: Movesmrg (if RPP_MODE=Y).

The output file created by Met4moves program for use by Movesmrg. It contains gridded daily min/max temperatures for every inventory county. Met4moves adjusts the native MCIP time zone (GMT) to local time in order to properly use the SMOKE-ready lookup tables, which are given in local time.

Type: I/O API NetCDF - adapted gridded format

Used by: Smkmerge and Smkreport
