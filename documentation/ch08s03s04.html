8.3.4. PROCDATES: Procdates date list

SMOKE can process specific sequential or non-seqential days of the year. The PROCDATES file is used to indicate which days should be processed temporally. The format of this file is shown in Table 8.36, “Format for PROCDATES”.

Table 8.36. Format for PROCDATES

Line Position Description
1+ A Seven digit Gregorian date (YYYYMMDD) (required)
  B Six digit Start time (HHMMSS) (required)
  C Six digit Run length (HHMMSS) (required)
