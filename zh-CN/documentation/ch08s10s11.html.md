### []{#sect_input_sccxref}8.10.11. `SCCXREF`{.envar}: Reference SCC input file {#sccxref-reference-scc-input-file .title}

</div>

</div>
:::

This file provides mapping between reference SCC (i.e., SCC8) and full
SCC (i.e., SCC10) for **Smkinven** to convert reference SCC-level
inventory to full SCC inventory:

::: {.table}
[]{#tbl_input_sccxref}

**Table 8.97. Delimited format for `SCCXREF`{.envar}**

  Lines   Position   Description
  ------- ---------- ---------------------------------------------
  1+      A          Full SCC (20-character maximum string)
          B          Reference SCC (20-character maximum string)
:::

::: {.example}
[]{#d0e52180}

**Example 8.25. `SCCXREF`{.envar} example file**

``` {.programlisting}
Full_scc,Reference_scc
"2201110262","2201110200"
"2201110281","2201110200"
"2201110362","2201110300"
"2201110381","2201110300"
"2201110462","2201110400"
"2201110481","2201110400"
"2201110562","2201110500"
"2201110581","2201110500"
"2201210262","2201210200"
"2201210281","2201210200"
....
```
